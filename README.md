# DocSpec test documents

A collection of openly-licensed documents for testing [DocSpec](https://github.com/docspec/docspec), a document converter.

## What is this?

This repository serves as a test corpus for DocSpec, collecting real-world documents in various formats to ensure robust document conversion. All documents must be openly licensed and properly attributed.

## How to donate a document

The easiest way is to [**create an issue**](../../issues/new/choose) using the "Document donation" template. Just fill out the form with your document's information, and we'll handle the rest!

Alternatively, you can submit a pull request (see [CONTRIBUTING.md](CONTRIBUTING.md)).

## Requirements for Contributors

If you're submitting a pull request:

- **Git LFS** is required for binary formats (DOCX, ODT, PDF, EPUB, etc.)
- Install: `git lfs install`
- LFS is automatically configured via `.gitattributes`

## Repository Structure

Documents are organized by format and group:

```
documents/<format>/<group>/<filename>.<ext>

Examples:
  documents/docx/pandoc/headings.docx
  documents/markdown/examples/sample.md
  documents/pdf/academic/paper.pdf
  documents/odt/libreoffice/newsletter.odt
```

All attribution and metadata is tracked in [`ATTRIBUTION.json`](ATTRIBUTION.json). The [`ATTRIBUTION.md`](ATTRIBUTION.md) file is autogenerated from the JSON.

## Accepted licenses

Documents must be under an open license that permits redistribution and modification:

- **Public Domain** (CC0, Unlicense)
- **Creative Commons**: CC-BY, CC-BY-SA, CC-BY-NC, CC-BY-NC-SA
- **GPL/LGPL/AGPL**
- **MIT, Apache-2.0, BSD**
- Other OSI-approved or FSF-approved free licenses

## What we're looking for

Diverse documents with interesting features:

- Complex formatting (tables, images, footnotes, equations)
- Different languages and scripts
- Edge cases and unusual structures
- Various file formats
- Real-world examples
